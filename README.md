# Flexible Project

This project contains:
- `react-app`: React frontend (Vite)
- `dotnet-app`: ASP.NET Core Web API backend

## Getting Started

### React Frontend
1. `cd react-app`
2. `npm install`
3. `npm run dev`

### .NET Backend
1. `cd dotnet-app`
2. `dotnet run`
